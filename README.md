# BANK-CUSTOMER-SEGMENTATION-ANALYSIS-Power-BI
Bank Customer Segmentation &amp; Analysis using Power BI for RFM-based insights, dashboards, and business decision support.


## Project Overview
This project focuses on **analyzing bank customers** to provide actionable business insights. Using **Power BI**, the project implements **RFM (Recency, Frequency, Monetary) segmentation** to categorize customers into Loyal, New, and Lost groups, helping the bank **reduce churn** and optimize marketing strategies.

The project demonstrates skills in **data modeling, DAX optimization, interactive dashboards, and KPI visualization** to support decision-making.

---

## Tools & Technologies
- **Power BI** – Dashboard development, visualization, and reporting  
- **DAX** – Calculated measures and optimized formulas  
- **Data Modeling** – 6-table relational model for performance optimization  
- **Excel / CSV** – Data preparation and cleaning  

---

## Key Features
- Multi-page **interactive dashboard** for analyzing 10,000+ customer records  
- **RFM segmentation** for customer categorization  
- **Optimized data model**, reducing dashboard load time by 30%  
- **Automated refresh cycles** and optimized DAX measures for 40% faster reporting  
- Delivered **12+ actionable business insights** on customer behavior and cross-selling opportunities

## Business Insights
Some of the insights derived from this project include:

- Cross-selling opportunities based on customer behavior patterns
